# BRImage-UI
UI for the BRImage Servlet.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploys to S3
```
npm run deploy
```
Precondition: correctly configured awscli profile for your AWS account
